# GIT PUSH SCRIPT

The Git Push Script (also known as `gitPush.py`) is very simple to use.

The only requirements for it are Python and Git.

To use it, simply open your terminal and run `python gitPush.py`

The script will than ask you for a commit message.

Make sure your message is formated like this: "Message"

You NEED the quotation marks or else the script won't work!
